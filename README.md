# Apoorv Darshan — Profile

A personal profile and portfolio for [Apoorv Darshan](https://github.com/apoorvdarshan), built from his public GitHub and LinkedIn presence.

## Run locally

```bash
npm install
npm run dev
```

The site runs at [http://localhost:3000](http://localhost:3000).

## Checks

```bash
npm run lint
npm run build
```
